**<b>**

用于显示粗体文字，不建议使用

**<base>**

用于指定文档中所有包含相对url的根url

```html
<base target="_blank" href='https://a.b.cc' />
<!-- 此条件下的 a 标签的行为可能会有所不同，如下面的例子 -->
<a href='#id'>go ID</a>
```

**<bdi>**

将内部元素的文字显示方向与父元素隔离开

```html
<p dir="ltr">
    This arabic word <bdi>ARABIC_PLACEHOLDER</bdi> is automatically displayed right-to-left.
</p>
<!-- This arabic word REDLOHECALP_CIBARA is automatically displayed right-to-left. -->
```

**<bdo>**

改写了文本的方向性, 使文本以不同的方向渲染呈现出来

**<blockquote>**

表示其内部的文本是引用来的
