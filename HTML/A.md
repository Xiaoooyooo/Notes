**<abbr>**

简写

```html
<abbr title='Hello World'>hello</abbr>
<!-- 鼠标悬停时会显示title中的信息 -->
```

**<address>**

其中的html用于向用户提供作者的相关联系信息

```html
<address>
	<a href='mailto:xxxxx@gmail.com'>邮件</a>
    <a href='tel:xxxxxxxxx'>电话</a>
</address>
```

**<area>**

与<img><map>一起使用，来定义一个图像映射

```html
<img src='xxxxx' alt='picture'/>
<map>
	<area shape='circle' coords='100,100,50' href='xxxxxx'/>
</map>
```

**<article>**

用于表示文档结构中的可独立分配或可复用的结构，如论坛帖子、文章、杂志、用户评论等

```html
<article>
	<h1>This is a title</h1>
    <p>
        This is a paragraph.
    </p>
</article>
```

**<aside>**

可以被单独的拆分出来而不使页面整体受影响，通常表现为侧边栏或标注框

```html
<aside>
	<!-- 这是一个侧边栏 -->
</aside>
```

**<audio>**

音频组件
