**<s>**

用删除线渲染文本，表示不再相关或不再准确

若要表示文本编辑请用<del>和<ins>

**<samp>**

用于标识计算机输出

```html
<p>
    Regular text,<samp>this is sample text,</samp> regular text.
</p>
```

**<script>**

用于嵌入或引用的可执行脚本

**<section>**

文档中独立的一个部分，一般包含一个标题

**<select>**

表示一个提供选项菜单的控件

```html
<select>
    <optgroup label='one'>
    	<option value='1'>1</option>
    </optgroup>
    <optgroup label='two'>
    	<option value='2'>2</option>
        <option value='3'>3</option>
    </optgroup>
</select>
```

**<slot>**

`Web Components`技术套件之一，充当占位符使用

**<small>**

表示注释和附属细则，包含版权和法律文本

**<source>**

为<audio>, <video>, <picture>指定多个媒体资源

**<span>**

短语内容通用容器，无任何特殊语义，类似于<div>，但<div>是块元素

**<strong>**

表示文本十分重要

**<style>**

包含文档的样式信息

**<sub>, <sup>**

<sub>表示文字下标，<sup>表示文字上标

**<summary>**

\<details>元素的总结内容

```html
<details>
	<summary>This is a summary.</summary>
    This is main content.
</details>
```
