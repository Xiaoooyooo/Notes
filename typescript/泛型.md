# 泛型

当在定义函数时，某些情况我们并不能确定函数的传入参数或返回值数据类型，这时候我们可以用泛型来表示参数或返回值的数据类型，此时定义的函数的参数数据类型会根据实际传入的参数而定。

```tsx
function fn<T>(a:T):T{
    return a
}

//一个函数内定义多个泛型
function fn2<a:T, b:K>(a,b):T|K{
    return Math.random() < 0.5 ? T : K
}

//调用函数时可以直接调用
fn('aaa')
fn(666)
//也可在调用时说明参数类型
fn2<string, number>('hello', 666)
```

泛型的定义可以继承自一个类或接口

```tsx
interface Inter {
    name: string
}
function fn3<T extends Inter>(a: T):T {
    return a.name
}
fn3('tom')	//出错
fn3({info: 'tom'})	//出错
fn3({name: 'tom'})	//正确
```
