# 二叉树

二叉树做为一种常用的数据结构，如字面意思，它的一个父节点最多只能同时拥有左右两个子节点。

+ 满二叉树：一颗二叉树有且仅有度为0或度为2的结点，并且度为0的结点在树的同一层上。
+ 完全二叉树：一棵深度为k的有n个结点的二叉树，对树中的结点按从上至下、从左到右的顺序进行编号，如果编号为i（1≤i≤n）的结点与满二叉树中编号为i的结点在二叉树中的位置相同，则这棵二叉树称为完全二叉树

## 二叉树的创建（以二叉排序树为例）

二叉排序树又称二叉查找树，亦称二叉搜索树，其定义如下

+ 若左子树不空，则左子树上所有结点的值均小于它的根结点的值；
+ 若右子树不空，则右子树上所有结点的值均大于它的根结点的值；
+ 左、右子树也分别为二叉排序树；
+ 没有键值相等的结点。

代码实现如下：

```tsx
class BinarySortTreeNode {
    value: number;
    left: BinarySortTreeNode | null;
	right: BinarySortTreeNode | null;
	constructor(value: number){
        this.value = value
        this.left = null
        this.right = null
    }
}
class BinarySortTree {
    root: BinarySortTreeNode | null;
	constructor(root: BinarySortTreeNode | null){
        this.root = root
    }
	insertNode(value: number):void{
        let newNode = new BinarySortTreeNode(value)
        if(this.root === null){
            this.root = newNode
        }else{
            this._insert(this.root, newNode)
        }
    }
    private _insert(currentNode: BinarySortTreeNode, newNode: BinarySortTreeNode):void {
        if(currentNode.value > newNode.value){
            if(currentNode.left === null){
                currentNode.left = newNode
            }else{
                this._insert(currentNode.left, newNode)
            }
        }else{
            if(currentNode.value < newNode.value){
                if(currentNode.right === null){
                    currentNode.right = newNode
                }else{
                    this._insert(currentNode.right, newNode)
                }
            }
        }
    }
}
```
