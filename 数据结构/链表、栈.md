# 链表、栈

## 链表

一种线性数据结构，其中的数据只能由上一个到下一个依次访问

### 单向链表

```js
class ListNode {
    constructor(value){
        this.value = value
        this.next = null
    }
}
const Head = new ListNode(-1)
let currNode = Head
for(let i = 0; i < 10; i++){
    currNode.next = new ListNode(i)
    currNode = currNode.next
}
console.log(Head)
```

### 双向链表

链表中的每一项既有next指向下一项，也有prev指向上一项

### 循环链表

即链表的最后一项的next指向该链表的第一项

## 栈

一种特殊的线性表，数据的存取遵循先进后出，后进先出的原则

### 顺序栈

可以用数组的方式表示

```js
class Stack{
    stack:Array<number>;
    constructor(){
        this.stack = []
    }
    push(value: number){
        this.stack.push(value)
        return this.length()
    }
	pop(){
        return this.stack.pop()
    }
	length(){
        return this.stack.length
    }
}
// 实例：括号匹配问题
```
