# 前端性能优化

目的：加快页面加载，提升网页响应速度，从而提高用户体验。

## 尽量减少HTTP请求

1. 资源合并与压缩

   尽可能将外部引入的css、js文件进行合并，并通过工具进行压缩。

2. 合并CSS图片

   将不同的图片整合到一张大的图片中。

3. 使用网页内联图像

   体积较小小的图片可以使用Base64格式

4. 图片懒加载

   仅在图片处于浏览器视图窗口时才请求图片

5. 合理利用浏览器缓存

## html文件资源引用

1. 将外部引入的脚本置于网页底部或者使用defer属性
   + defer：文档加载与js下载同时进行，但js只会在文档加载完毕之后执行
   + async：文档加载与js下载同时进行，但js下载完毕后会立即执行，从而阻塞文档的加载

2. 将css文件的引入至于head标签中

   这样浏览器在构建DOM树的过程中同时能进行页面的渲染，使屏幕尽快呈现内容

3. 按需加载js文件

4. 使用内容分发网络（CDN）
