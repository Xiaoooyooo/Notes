1. **word-bread**
+ `normal:`英文字符只在空白处断行，中文字符任意断行；
   + `break-all:`字符可在任意处断行；
   + `keep-all:`字符不断行；
   + `break-word:`英文字符首先在空白处断行，”迫不得已“在一个单词间断行，中文任意断行。
2. **overflow-wrap**

   + `normal:`同`word-break: normal`
   + `break-word:`同`word-break: break-word`
3. **letter-spacing**

   + `normal:`依据文字的默认对齐方式
   + `[length]:`手动控制字符剑距
4. **line-break**

   + 用来处理如何断开（break lines）带有标点符号的中文、日文或韩文（CJK）文本的行
5. **paint-order**

   + 控制文本区域和图形绘制的填充和绘制（和markers）的顺序
   + `normal:`同`fill stroke markers`
   + `fill:`同`fill stroke markers`
   + `stroke:`同`stroke fill markers`
   + `markers stroke fill:`先 markers, 然后 stroke, 然后 fill
   + 以此类推
6. **tab-size**

   + 自定义制表符（U+0009）的宽度
   + `[integer]:`给定个数的空格符的宽度

   + `[length]:`给定长度
7. **text-align**

   + 控制元素内部行内内容对齐方式
   + `start:`如果内容方向为从左至右，则同`left`，反之同`right`
   + `end:`如果内容方向为从左至右，则同`right`，反之同`left`
   + `left:`左对齐
   + `right:右对齐`
   + `justify:`两侧对其（除最后一行）
8. **text-align-last**

   + 控制文本最后一行的对齐方式，属性同`text-align`
9. **text-indent**
+ 定义块元素首行文本的缩进
   + `[length]:`
   + `[percentage]:`块元素宽度的百分比
   + `each-line:`Chrome未支持
   + `hanging:`Chrome未支持
10. **text-justify**
    + 定义当`text-align`设置为`justify`时的对齐方式
    + `none:`表现为禁用`text-align: justify`
    + `auto:`默认值，浏览器自动判断
    + `inter-word:`在单词间添加空格来实现两端对齐，会影响`word-spacing`的值
    + `inter-character:`在文本字符间添加空格实现两端对齐，会影响`letter-spacing`的值
11. **text-size-adjust**
    + 控制一些手机或平板设备上使用的文本溢出算法，其他类型的设备上的浏览器会忽略此属性
    + `none:`禁用文本溢出算法
    + `auto:`启用文本溢出算法
    + `[percentage]:`启用文本溢出算法，并用一个百分比来确定文本放大程度
12. **text-transform**
    + 指定文本的表现方式
    + `capitalize:`强制每个单词的首字符大写
    + `uppercase:`强制每个字符大写
    + `lowercase:`强制每个字符小写
    + `none:`按文本的原本形式显示
    + `full-width:`强制每个字符写进一个方形里（与中文或日文字符对齐）
13. **white-space**
    + 定义文本内空白符的处理方式
    + `normal:`合并多余的空白符，允许换行，删除行首行尾空格符
    + `nowrap:`合并多余空百符，不允许换行，删除行首行尾空白符
    + `pre:`保留多余空白符，仅在遇到换行符或`<br />`元素时换行
    + `pre-wrap:`保留多于空白符，允许换行，保留行首行尾空白符
    + `pre-line:`合并多余空白符，允许换行，删除行首行尾空白符
    + `break-spaces:`保留多余空白符，允许换行，保留行首行尾空白符

1. **目前支持浏览器较少的属性**
   + `hanging-punctuation`
   + `hyphens`
